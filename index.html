📋 Aide-mémoire réglementation ▼

⭐ Alcool — seuils et sanctions

  • Contravention : 0,20 à 0,79 g/sang → 135€ + 6 pts retrait
  • Délit : à partir de 0,80 g/sang → 4500€ max + 2 ans prison
  • Alcool + stupéfiants : 9000€ + 3 ans prison
  • Cannabis seul : 2× plus de risque d'accident
  • Cannabis + alcool : 29× plus de risque

⭐ Dépassement — distances latérales

  • Piéton/2 roues en agglo : 1 m minimum
  • Piéton/2 roues hors agglo : 1,50 m minimum
  • Distance avant de se rabattre : 50 m minimum
  • 3ème voie autoroute interdite pour véhicule >3,5t ou >7m

⭐ Porte-à-faux (PAF)

  • PAF avant (moyenne) : 2,50 m
  • PAF arrière (moyenne) : 3,50 m
  • Je tourne à DROITE → PAF arrière se déporte à GAUCHE

⭐ Systèmes de sécurité

  • ABS/ABR = Anti Blocage des Roues
  • ESP = Stabilisateur Électronique Programmable
  • AFU = Aide au Freinage d'Urgence
  • ASR = Anti-patinage des roues
  • EAD = Éthylotest Anti-Démarrage

⭐ Passages à niveau

  • 15 400 PN en France
  • 98% des accidents = non-respect Code de la route
  • Type le plus fréquent : demi-barrières automatiques
  • Urgence sur voies : évacuer passagers + poste secours PN

⭐ Éco-conduite

  • Objectifs : économie carburant, moins d'usure, réduire pollution
  • Changer de vitesse à bas régime
  • AD Blue = antipollution pour diesels récents
  • Norme Euro 6 = directive réduction pollution constructeurs

⭐ Durées de conduite (RSE)

  • Continue : 4h30 · Nuit : 4h
  • Journalière : 9h (max 10h × 2/semaine)
  • 1 semaine : 56h · 2 semaines : 90h

⭐ Pauses

  • Continue : 45 mn après 4h30
  • Fractionnée : 15 mn puis 30 mn

⭐ Repos

  • Journalier normal : 11h
  • Journalier fractionné : 3h + 9h = 12h total
  • Repos réduit : 9h
  • Hebdomadaire non-réduit : 45h
  • Amplitude max : 14h

⭐ Chronotachygraphe numérique

  • Contrôle technique : tous les 2 ans
  • Panne → réparation : 7 jours maxi
  • Perte / vol carte : 15 jours de conduite sans
  • Transfert données CARTE : 56 jours maxi
  • Transfert données CHRONO : 90 jours maxi
  • Transfert par EMPLOYEUR : 28 jours maxi
  • Archivage : 1 an

⭐ Majoration PTAC ralentisseur

  • Ralentisseur 400 kg → +400 kg
  • Ralentisseur 600 kg → +500 kg

⭐ Poids voyageurs

  • Autobus : 65 kg · Autocar : 70 kg

⭐ Contrôle technique TEC (tous les 6 mois)

  • Lettre A = Accepté
  • Lettre S = Sursis → contre-visite sous 1 mois
  • Lettre R = Refusé (interdiction de circuler)

⭐ Visites médicales conducteurs

  • Jusqu'à 60 ans : tous les 5 ans
  • À partir de 60 ans : tous les ans

⭐ Barrière de dégel

  • TC à vide sur barrière 12T : OUI
  • Mention "demi-charge" : NON

⭐ Vitesses clés

  • Brouillard <50m → 50 km/h
  • Agglo portion 70 → 70 km/h
  • Arrière autobus → 70 km/h
  • Pluie route prioritaire → 80 km/h
  • Arrière autocar → 90 et 100 km/h
  • Limiteur / autoroute → 100 km/h
📌 Consignes d'utilisation ▼
  1. Vous pouvez sélectionner Tout mélangé ou choisir l'un des 6 modes, puis lancer le quiz.
  2. Pour la signalisation, si vous avez un doute, le numéro de la fiche est affiché en bleu au-dessus de la question. Vérifiez ensuite sur le document papier.
  3. Pour réinitialiser toutes les statistiques, cliquez sur « Réinitialiser toute la progression ». Pour réinitialiser un seul mode, utilisez la section « Réinitialiser les stats d'un mode » et choisissez le mode concerné.
  4. La lecture vocale est désactivée par défaut. Pour l'activer, lancez un quiz et cliquez sur le bouton Lecture vocale au-dessus de la question jusqu'à ce qu'il devienne vert.
  5. Dans le quiz, vous pouvez activer la barre de progression en cliquant sur « BARRES DE PROGRESSION ». Elle vous indique si votre réponse à la question précédente était correcte. Conseil : désactivez-la quand vous vous sentez prêt afin de vous placer dans les conditions d'un examen.
  6. Dans « Aide-mémoire réglementation », vous trouverez les principaux points clés à connaître pour les fiches.
